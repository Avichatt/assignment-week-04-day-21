# Week 4 Day 21 Assignment

This repository contains two parts of the NumPy and Machine Learning Basics assignment, developed as part of the data science curriculum.

## Project Structure

### Part 1: NumPy Basics (AM Session)
The `part-*-am.py` files demonstrate fundamental NumPy skills:
- **Concept Application**: Array creation, indexing, slicing, and statistical operations.
- **Stretch Problem**: Matrix operations, solving linear equations, and a performance comparison between loops and vectorization.
- **Interview Readiness**: Explanations of broadcasting and vectorization.

### Part 2: Machine Learning Fundamentals (PM Session)
The `part-*-pm.py` files cover foundational ML concepts using pure NumPy:
- **Concept Application**: Synthetic dataset generation, training basic models, and manual calculation of MSE/Accuracy.
- **Stretch Problem**: Simulation of the Bias-Variance tradeoff with polynomial regression and U-curve visualization.
- **Interview Readiness**: Explanations of regression vs. classification and the bias-variance tradeoff.

## Setup
Ensure you have Python installed. The scripts use `numpy` and `matplotlib`.

```bash
pip install numpy matplotlib
```

## Running the Scripts
Each script is designed to be standalone. Example:
```bash
python part-a-am.py
```
